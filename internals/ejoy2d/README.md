EJOY 2D
=======

Make
====

For Windows and msvc

* msvc\make.bat
* ej2d examples/ex01.lua to test

For Windows and mingw32

* Install glew 1.9
* make or make mingw
* ej2d examples/ex01.lua to test

For Linux ,

* Install glew 1.9
* Install freetype 2
* make or make linux
* ./ej2d examples/ex01.lua to test

For Mac OS ,

* Install glfw3
* Install freetype 2
* make or make macosx
* ./ej2d examples/ex01.lua to test
